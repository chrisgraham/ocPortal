Thank you for downloading ocPortal.

To install ocPortal onto your web server, upload the two files in this directory (data.ocp and install.php) to your web server, and then run the install.php script by navigating to it in your web browser.

For more detailed instructions, please see our installation tutorial: http://ocportal.com/docs/tut_install.htm

For more information about ocPortal, visit our website: http://ocportal.com/
